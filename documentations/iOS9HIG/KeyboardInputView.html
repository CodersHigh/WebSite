Keyboards and Input Views

적당하다면, 시스템이 제공하는 화면상의 키보드를 대체할 커스텀 입력 뷰를 디자인 할 수 있다.

커스텀 입력 뷰를 제공한다면, 그 기능이 사람들에게 명백하도록 하라.

또한 키보드의 위쪽(또는 당신의 커스텀 입력 뷰)에 나타나는 분리된 뷰인, 커스텀 입력 액세서리 뷰를 제공할 수 있다.

사람들이 당신의 입력 뷰안에 있는 커스텀 컨트롤을 탭할 때 표준 키보드 클릭 사운드를 사용해서 청각적인 피드백을 제공하라. 코드로 사운드를 켜는 방법은 UIDevice 클래스 레퍼런스의 playInputClick문서를 보라.

Note

표준 클릭 사운드는 화면상에 나타난 커스텀 입력 뷰에만 사용할 수 있다. 사람들은 Settings > Sounds 에서 - 당신의 커스텀 입력 뷰에서 오는 것을 포함한 - 모든 키보드 클릭을 끌 수 있다.
