애플리케이션 프로토타이핑 과정

애플리케이션 프로토타이핑 과정은 앱의 아이디어를 구체화하고 UI 디자인을 만드는 과정입니다.

코더스하이에서 제공하는 프로토타입 과정은 아래와 같은 단계를 거칩니다.

ADS - 목업스케치 - Xcode 스토리보드 - 오토레이아웃 - 파일럿플랜트

ADS

Application Definition Statement 는

목업스케치

Xcode 스토리보드

오토레이아웃

파일럿플랜트

코딩 경험이 없는 경우, 먼저 앱 개발을 위한 인터페이스를 디자인하는 것을 배우는게 중요합니다.

애플리케이션 프로토타이핑 과정에서는
애플의 iOS Human Interface Guideline을 학습한 뒤, 디자인 씽킹을 통해 문제를 정의하고 앱 프로토타입을 만듭니다.

프로토타이핑의 반복
손 그림 프로토타입 단계를 거쳐 실제 개발자들이 사용하는 IDE(통합개발환경)인 Xcode의 스토리보드 기능을 이용해 실제 앱에 가까운 프로토타입을 제작하게 되며, 코딩을 제외한 모든 부분에서 개발자에 가까운 지식을 가지게 됩니다.
Swift iOS 코딩 과정으로의 진입 (이미지)
프로토타입 제작을 통해 개발 방식과 개발자 도구를 경험해 보았다면 실제 앱을 코딩하는 일은 훨씬 수월해 집니다. Swift iOS 코딩 과정을 통해 본인의 앱을 직접 만들어 보세요.

커리큘럼
섹션 시간 강의내용
아이디어
정리하기		 16H Product - ADS - MentalModel - UI Sketch
애플리케이션의 아이디어를 ADS로 정의하고, 사용자 분석을 한다.

HIG - UI 디자인의 기본 - 디자인 전략 - UI요소 - 아이콘과 이미지 디자인
애플리케이션 사용자 인터페이스의 표준 가이드라인을 통해 모바일 앱의 특성을 파악한다.
개발도구
익히기		 18H IDE - Xcode Storyboard
스토리보드의 구성요소와 사용법을 안다

Layout - Auto Layout
제약사항 기반의 레이아웃 기술 방식인 오토레이아웃의 개념을 알고 사용법을 익힌다.

Tool - Git & GitHub
문서의 버전관리 시스템인 Git에 대한 개념을 잡고, Git를 편리하게 사용할 수 있는 GitHub 사이트를 이용해 본다.
프로토타입
만들기		 16H Tool - PilotPlany Library
프로토타입 도구인 PilotPlant의 사용법을 배우고 프로토타입을 구현하는 데 활용한다.

Presentation - Keynote
완성된 프로토타입을 이용해 발표를 함으로서 커뮤니케이션 수단으로서의 프로토타입을 활용해본다.
