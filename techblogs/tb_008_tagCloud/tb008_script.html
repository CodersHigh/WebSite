3d 스크롤이 가능한 태그 클라우드 만들기

Posted: August 7, 2018. | By: 김소연 (github.com/aa9390)

이번 포스트에서는 다음 사진과 같이 위/아래/좌/우로 3d 스크롤이 가능한 태그 클라우드를 구현해 보겠습니다.
동작 방식은 맨 처음 구름 버튼을 클릭하면 관련된 구름 버튼들이 주위에 나타나고, 배경 클릭 시 해당 구름들이 사라지게 할것입니다. (작성중 - 스크린샷은 다른 것으로 교체 예정)
본 프로젝트는 외부 라이브러리를 사용했습니다. (바로가기)

---- 스토리 보드 스크린샷 -----

우선 single view app으로 새로운 Xcode 프로젝트를 생성합니다. 이 프로젝트에서는 총 2개의 View Controller들을 사용하겠습니다. 각각의 View Controller들은 다음과 같은 기능을 가지고 있습니다.


[메인 화면] : 태그 클라우드를 실제로 구현하는 뷰로, (작성중)

[상세정보 화면] : 프로젝트 구름 버튼 클릭 시 이동하는 상세 정보 뷰 입니다.

CocoaPods는 Swift 및 Objective-C 코코아 프로젝트의 종속성 관리자입니다.
28,000 개가 넘는 라이브러리를 가지고 있으며 170 만 개가 넘는 응용 프로그램(앱)에서 사용되고 있습니다.
CocoaPod은 프로젝트를 우아하게 확장 할 수 있도록 도와줍니다.
